Links e Âncoras

Links externos

Os hyperlinks permitem que você conecte diferentes pontos na World Wide Web. Toda vez que você acessa um site ou clica em um local para outra página, outro site ou para baixar um arquivo, você está interagindo com um hyperlink

Criando um link

Acesse a página do GE: clique aqui

Parâmetro target controla o comportamento do site após interação com hyperlink

  • _blank: abre o link em uma nova janela
  • _self: abre o link na própria aba

Por exemplo, o perfil do GitHub abrirá em uma outra janela: Clique aqui para o perfil do GitHub

Links internos

Levam o visistante a outra página dentro do nosso próprio site.

Acesse aqui a próxima página

Podemos utilizar o atributo "rel" na tag <a> para indicar o que significa o link

  • next: link é para a próxima parte do documento atual
  • prev: link é para a parte anterior do documento atual
  • author: link é para o site do autor do documento atual
  • external: link é para um site que não faz parte do documento atual
  • nofollow: link é para site não endossado, como um link pago

        Continuar navengando no site: <a href = "pagina2.html" target="_self" rel="next"> </a>
        Abrir o GitHub em uma nova janela: <a href="https://gustavoguanabara.github.io" target="_blank" rel="external"></a>

Links para downloads

Link diretamente para o arquivo que se deseja efetuar o download com os seguintes atributos:

  • download: valor para o nome do arquivo a ser baixado
  • typpe: indica ao navegador o tipo de arquivo a ser baixado.

Media types mais utilizados:

  • application/zip
  • application/pdf
  • text/html
  • video/mp4
  • image/png
