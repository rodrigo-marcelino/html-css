Listas ordenadas

São as listas em que a ordem dos itens importam. A tag ol apresenta o parâmetro type para configurar o tipo de marcador e o parâmetro start para identificar o marcador de início

  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Listas não ordenadas

A ordem dos itens não é importante

  • Feijao
