Para que servem

As tags tem função de agrupar conteúdos. No HTML4, tinha apenas as tags div (block-level) e span(inline-level) para inserir caixas.

Hoje, temos as Semantic Tags que organizam nosso site pelo significado do contéudo (das caixas)

As Semantic Tags ajudam o nosso site a ser melhor vistos para os navegadores

Abaixo, temos uma foto de exemplo para a organização de um site

  • Header: <h1> - <h1>
  • Nav: possui links de navegação pelo website: menu
  • Main: delimita o conteúdo principal do nosso site
  • Section: agrupamento temático de conteúdos, tipicamente com um cabeçalho
  • Article: conteúdo que pode ser lido de forma independente
  • Aside: Delimita um conteúdo periférico e complementar ao conteúdo principal de um artigo ou seção.
  • Footer: rodapé
